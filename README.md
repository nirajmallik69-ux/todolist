# 📝 Smart To-Do List

A modern and responsive **Smart To-Do List** web application built using **HTML, CSS, and JavaScript**. It helps users manage daily tasks with a clean Glassmorphism UI, Dark Mode, and Local Storage support.

##  Features

-  Add new tasks
-  Mark tasks as completed
-  Delete tasks
-  Save tasks using Local Storage
-  Dark Mode with theme persistence
- Remaining & Total task counter
-  Fully Responsive Design
-  Glassmorphism UI
- Smooth animations and transitions
- Press **Enter** to add tasks

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API
